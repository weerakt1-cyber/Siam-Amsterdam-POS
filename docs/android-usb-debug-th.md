# วิธีแก้ปัญหา: เครื่อง Android เสียบ USB แล้ว adb ไม่เจอ

**อาการ:** เสียบสาย USB แล้ว Windows เห็นเครื่อง (SAMSUNG Mobile USB Composite Device — Status: OK)
แต่ `adb devices` ไม่ขึ้นรายการเครื่องเลย

**สาเหตุที่พบบ่อยที่สุด:** หน้าจอเครื่องล็อกอยู่ตอนที่ป๊อปอัพ "อนุญาตแก้ไขข้อบกพร่องผ่าน USB" (Allow USB debugging) ขึ้นมา ทำให้พลาดกดอนุญาต หรือ USB debugging ถูกปิดไปแล้ว

---

## ขั้นตอนที่ 1 — ปลดล็อกหน้าจอแล้วเช็คป๊อปอัพ

1. **ปลดล็อกหน้าจอโทรศัพท์/แท็บเล็ต** ให้เรียบร้อยก่อน (ต้องปลดล็อกค้างไว้ ป๊อปอัพถึงจะโต้ตอบได้)
2. มองหาป๊อปอัพที่ขึ้นข้อความประมาณนี้:
   > **อนุญาตแก้ไขข้อบกพร่องผ่าน USB?**
   > (Allow USB debugging?)
   > ลายนิ้วมือ RSA คีย์ของคอมพิวเตอร์คือ: ...
3. **ติ๊กถูก** ที่ช่อง "**อนุญาตจากคอมพิวเตอร์เครื่องนี้เสมอ**" (Always allow from this computer)
4. กด **อนุญาต** (Allow)

**ถ้าไม่เห็นป๊อปอัพเลย:**
- ถอดสาย USB ออก แล้วเสียบใหม่อีกครั้ง (การถอด-เสียบใหม่จะทริกเกอร์ป๊อปอัพนี้ขึ้นมาอีกรอบ)
- รอสัก 2-3 วินาทีหลังเสียบ แล้วดูหน้าจอโทรศัพท์

---

## ขั้นตอนที่ 2 — เช็คว่า USB Debugging ยังเปิดอยู่ไหม

บางครั้ง Android จะปิด USB debugging เองอัตโนมัติเมื่อรีสตาร์ทเครื่อง หรือหลังไม่ได้ใช้งานนานๆ

1. ไปที่ **การตั้งค่า (Settings)**
2. เลื่อนหา **ตัวเลือกสำหรับนักพัฒนา (Developer options)**
   - ถ้าไม่เจอเมนูนี้ ให้ไปที่ **เกี่ยวกับโทรศัพท์ (About phone)** แล้วกดที่ **หมายเลขบิลด์ (Build number)** ติดต่อกัน 7 ครั้ง จะมีข้อความแจ้งว่า "คุณเป็นนักพัฒนาแล้ว" (You are now a developer)
3. เข้า **ตัวเลือกสำหรับนักพัฒนา** แล้วเช็คว่า **แก้ไขข้อบกพร่องผ่าน USB (USB debugging)** เปิดอยู่ (toggle เป็นสีเขียว/เปิด)
4. ถ้าปิดอยู่ ให้เปิดขึ้นมา แล้วกลับไปทำขั้นตอนที่ 1 อีกครั้ง (ถอด-เสียบสายใหม่)

---

## ขั้นตอนที่ 3 — เช็คโหมดการเชื่อมต่อ USB

เครื่อง Samsung บางรุ่น ถ้าตั้งโหมด USB เป็น "ชาร์จเท่านั้น" (Charging only) จะไม่เปิดช่องทาง ADB ให้ใช้

1. ดึงแถบการแจ้งเตือนลงมา (swipe ลงจากด้านบนจอ)
2. มองหาการแจ้งเตือนประมาณ **"กำลังชาร์จอุปกรณ์นี้ผ่าน USB"** หรือ **"แตะเพื่อดูตัวเลือกเพิ่มเติม"**
3. แตะเข้าไป แล้วเลือกโหมด **โอนไฟล์ (File Transfer / MTP)** หรือ **PTP** แทน **ชาร์จเท่านั้น (Charging only / No data transfer)**

---

## หลังทำครบทั้ง 3 ขั้นตอนแล้ว

บอกในแชทว่า "เสร็จแล้ว" หรือ "ลองใหม่ได้เลย" — จะรัน `adb devices` เช็คให้อีกครั้งว่าเครื่องขึ้นมาหรือยัง ถ้าขึ้นแล้วจะไปต่อขั้นตอน ติดตั้งแอปใหม่ (reinstall) แล้วดู log ตอนกด Scan หา Bluetooth printer

---

## ถ้ายังไม่ขึ้นอีก

ลองสิ่งเหล่านี้เพิ่มเติม:
- ลองเปลี่ยนสาย USB เส้นอื่น (สายชาร์จบางเส้นไม่มีสายข้อมูล มีแค่สายไฟชาร์จอย่างเดียว)
- ลองเสียบพอร์ต USB อื่นบนคอมพิวเตอร์ (บางทีพอร์ต USB 3.0/USB-C hub มีปัญหา)
- รีสตาร์ทโทรศัพท์/แท็บเล็ต 1 รอบ แล้วเสียบใหม่
